# Entomology Notes

This repository contains basic and advanced notes on entomology, including insect classification, morphology, and biodiversity concepts. It is intended for students and researchers in zoology.

Author: Dr. Santosh Kumar Charan
Assistant Professor, University of Rajasthan
## Topics Covered
- Insect Classification
- Morphology of Insects
- Types of Legs and Wings
- Economic Importance of Insects

## Purpose
This repository is created to provide easy and structured notes for MSc Zoology students.

## Future Work
More detailed notes, diagrams, and research-based content will be added regularly.
